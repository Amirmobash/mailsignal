# MailSignal — Next.js + Tailwind

## Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Images
The page currently loads the existing repository images directly from:
`https://raw.githubusercontent.com/Amirmobash/mailsignal/main/assets/images/`

For production, you can copy the images into `public/images/` and replace the remote URLs with local `/images/...` paths.
