# MailSignal Website

Statische Website für GitHub Pages.

## Upload auf GitHub
1. Inhalt dieses Ordners in dein Repository `mailsignal` kopieren.
2. In GitHub: Settings → Pages → Branch `main` und Folder `/root` auswählen.
3. Speichern. Danach ist die Website über GitHub Pages erreichbar.

## Dateien
- `index.html` Hauptseite
- `assets/css/style.css` Design
- `assets/js/main.js` Menü und Animationen
- `assets/images/` Bilder und Favicon
